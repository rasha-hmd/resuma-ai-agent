# Senior Full Stack Developer - FinTech Startup

**Company:** InnovatePay Technologies  
**Location:** San Francisco, CA (Hybrid)  
**Experience Level:** Senior (5+ years)  
**Employment Type:** Full-time  

## About InnovatePay Technologies

InnovatePay Technologies is a rapidly growing fintech startup revolutionizing digital payment solutions for small and medium-sized businesses. We're building the next generation of payment processing platforms that combine cutting-edge technology with exceptional user experience.

## Job Overview

We are seeking a highly skilled Senior Full Stack Developer to join our engineering team. You will play a key role in designing, developing, and maintaining our core payment platform, working with modern technologies and contributing to architectural decisions that will shape our product's future.

## Key Responsibilities

- Design and develop scalable web applications using React, Node.js, and TypeScript
- Build and maintain RESTful APIs and GraphQL services for our payment processing platform
- Implement secure, PCI-compliant payment systems and financial data handling
- Work with cloud technologies (AWS, Azure) to deploy and manage applications
- Collaborate with cross-functional teams including product managers, designers, and DevOps engineers
- Participate in code reviews, technical design discussions, and architecture planning
- Mentor junior developers and contribute to team knowledge sharing
- Implement automated testing strategies and maintain high code quality standards
- Optimize application performance and ensure scalability for growing user base
- Work in an agile environment with continuous integration and deployment practices
